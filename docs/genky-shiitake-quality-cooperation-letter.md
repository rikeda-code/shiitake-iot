# ゲンキー様向け椎茸不良の改善に向けたご協力のお願い

（近代青果様宛　社外送付用文書）

---

近代青果株式会社
ご担当者様

拝啓　時下ますますご清栄のこととお慶び申し上げます。平素より格別のお引き立てを賜り、厚く御礼申し上げます。

このたびは、弊社が納品いたしました椎茸につきまして、ゲンキー様店頭にて品質不良が確認され、近代青果様をはじめ関係の皆様に多大なご迷惑・ご心配をおかけしましたこと、心より深くお詫び申し上げます。誠に申し訳ございませんでした。

つきましては、再発防止に向けて原因究明を徹底し、品質改善に努めてまいりたく存じます。そのためには流通経路全体における椎茸の状態推移を把握する必要があり、近代青果様をはじめ関係各社様にご協力をお願いしたく、ご連絡差し上げました。

## ■ 調査の目的

近代青果様を経由してゲンキー様店頭に並ぶ椎茸について、どの工程で品質不良が発生しているか、また不良品がどのような状態であるかを把握し、今後の不良品対策・品質改善に役立てることを目的としております。

## ■ 現状の流通工程

① 椎茸収穫（弊社）　→　② 椎茸梱包（弊社）　→　③ 椎茸保管（弊社）　→
④ 近代青果様納品　→　⑤ 福井商事様集荷・配送　→　⑥ 大津市公設卸売市場納品　→
⑦ センター納品　→　⑧ 店舗納品・店頭販売

## ■ 弊社にて実施すること（①～③）

- ②梱包時点における商品状態の写真撮影
- 段ボールへの管理番号の付番による、商品状態推移の追跡
- ⑥⑦⑧の各工程における商品状態の確認（弊社社員が現地にて実施）

## ■ ご協力いただきたい内容

上記調査の実施にあたり、流通経路全体の窓口でいらっしゃる近代青果様に、福井商事様・ゲンキー様への調整および情報提供についてご協力を賜りたく存じます。

**1. 福井商事様へのご協力依頼（近代青果様よりお声がけをお願いいたします）**
集荷から大津市公設卸売市場への配送までの温度データ取得にご協力いただきたく存じます。弊社にて準備する温度計を段ボールに同梱いたしますので、大津市場到着後、弊社社員による回収にご協力いただけますと幸いです。

**2. ゲンキー様へのご協力依頼（近代青果様よりお声がけをお願いいたします）**
店頭での販売タイミングおよび対象店舗をお教えいただき、店頭に並ぶ商品の状況を弊社社員が確認させていただけますよう、ご協力をお願いいたします。

**3. 近代青果様へのお願い**
上記1・2について福井商事様・ゲンキー様へのご協力要請をいただくとともに、④～⑧の各ポイントにおける納品・集荷・配送等のタイミング（日時）について情報共有をお願い申し上げます。

## ■ 今後の進め方

具体的な実施時期・詳細につきましては、貴社および関係各社様のご都合を踏まえ、改めてご相談させていただきたく存じます。ご多忙のところ大変恐縮ではございますが、何卒ご協力賜りますようお願い申し上げます。
